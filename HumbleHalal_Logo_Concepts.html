6. Technical Specifications

Reference table for implementing the logo across digital and print.

Element Specification Notes
Icon Symbol Google Material Symbols "mosque" — Outlined, wght 300–400 Use FILL 0 for outlined, FILL 1 for filled variant
Icon Container Rounded square, radius = 25% of width 72px → 18px radius, 56px → 14px, 44px → 11px
Icon Gradient 135° linear: #047857 → #065f46 Subtle depth, never flat emerald
"Humble" Font Manrope ExtraBold (800), -0.02em tracking Always same weight, never thin
"Halal" Font Playfair Display Bold Italic (700i) Always italic, creates contrast with Manrope
Minimum Size Full lockup: 120px wide. Icon only: 16px Below minimum, use icon-only mark
Clear Space Minimum = height of icon container on all sides No other elements within clear space zone
File Formats SVG (web), PNG (social, 2x), PDF (print) Always export at 2x for retina displays
